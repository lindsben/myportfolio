My Projects:

To-Do List App Wireframe

The purpose of this project was to put my wireframe skills to use by creating my own version of a To-Do List app. I utilized my skills in Figma and my knowledge of React while creating this wireframe. One problem I encountered while creating this was coming up with a pleasant color scheme that goes along with the purpose of the app. I solved this by researching different color palettes and the emotional effects of certain colors. By doing this, I was able to come up with a simple palette that went along perfectly with my app wireframe.

Exercise App

I had never coded something like that before so it was definitely something to get used to. I did so by reading about React online to allow myself to become more knowledgable of the coding language as a whole.


Study Cards App

The purpose of this project was to once again utilize my skills and knowedge of Figma and React Native to create a wireframe for a Study Cards app. The main issue I had with this project was figuring out how to fit all of the components together on each screen while keeping the wireframe cohesive. Through trial and error, I was able to persevere past my struggles. I am really proud of my wireframe design for this project. I decided to make the wireframe a combination of a pretty blue and green color. I felt that these colors were perfect for an app that is "studious" in nature. I kept the fonts simple as well because I did not want to distract the user from why they are using the app, which is to study.

Quiz App
